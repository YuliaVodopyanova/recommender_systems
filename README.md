# recommender_systems

Репозиторий включает в себя файл типа .ipynb, содержащий реализации алгоритмов разного типа для создания рекомендательной системы для онлайн-кинотеатра. 

В файле описан используемый датасет, собраны шаги анализа и подготовки данных к их использованию для создания рекомендательной системы, а также непосредственные алгоритмы реализации рекомендательной системы:

* бейзлайны, основанные на популярности фильмов;
* контентная фильтрация с импользованием разных эмбеддингов;
* коллаборативная фильтрация;
* гибридная фильтрация, разными способами комбинирующая одиночные модели.
